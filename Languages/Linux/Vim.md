- Mode switching
i - insert mode (press esc to escape from the mode)
:w (for saving)
:q (for quitting the app)

- Movement
h-left	l-right
j-down	k-up

- Visual mode:
v - one character at a time
V - whole line

- Command mode:
yy - copies a line
yw - copies word
d - delete
dd - delete line
u - undo
dw - delete word forward
db - delete word backwardi
p - paste below
